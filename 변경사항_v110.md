# 변경사항 v110 — 원고↔사진 연동 + 게시별 성과 기록

배포본: `2026-07-19-ad-perf-v110` (v109 홍보 원고 생성기의 후속)

## 1. 📸 원고 ↔ 현장 사진 연동

[✍️ 원고 만들기] 모달 하단에 선택한 현장의 사진 현황이 붙습니다.

- 사진이 있으면: "📸 이 현장 사진 N장" + **[🖼 전/후 카드 만들기]**(기존
  전/후 갤러리로 바로 진입) + **[📂 사진 보기]**(사진 탭을 그 현장 필터로 열기)
- 사진이 없으면: 사진 탭에서 올려두라는 안내

원고 쓰다가 → 사진 고르러 가는 흐름이 끊기지 않습니다.

## 2. 📈 게시별 성과 기록

광고 관리 탭의 각 게시 기록에 **[📈]** 버튼이 생겼습니다.

- "이 글을 보고 온 문의 수" + 한 줄 메모(예: "욕실 전후 사진 반응 최고") 기록
- 문의가 있는 글은 배지(📈 4)로 표시
- 채널 화면 상단에 **이번 달 요약: 게시 n건 · 문의 m건** — 어떤 채널·형식이
  실제 손님을 데려오는지 데이터가 쌓입니다
- 입력 팁: 상담 때 "어디서 보고 연락 주셨어요?" 한마디면 채워집니다

데이터는 기존 `adPosts` 항목의 선택적 `res:{inq,memo}` 필드 —
**serializeData/applyData 스키마 변경 없음**(구 기록·구 버전과 하위호환).
