# 변경사항 v112 — 모바일·PC UI/UX 정비 (8건)

배포본: `2026-07-19-uxpass-v112` · 실화면 캡처 기반 검토(모바일 390px·PC 1440px)에서 나온 8건.

## 고침

1. **현장별 탭 빈화면 해소** — 파일이 0개면 등록된 현장(고객·단계·일정)까지
   숨겨지던 render 가드에서 현장별 탭 제외. 이제 폴더 연결 전에도 현장
   카드·진행 단계·액션이 그대로 보인다. (광고 탭 v109 수정과 동일 계열)
2. **탭별 빈 상태 문구** — 전 탭 공용이던 "파일을 끌어다 놓으세요"를 탭
   목적별로: 회계=홈택스 엑셀, 사진=현장 사진, 견적=견적서 엑셀, 서류=서류.
   모바일에 없는 "왼쪽에서" 표현도 PC/폰 구분 안내로 수정.
3. **대시보드 KPI 모순 해소** — 견적서 연결 전(매출 0)인데 수금·지출이
   있으면 "-마진·수금률 0%"로 숫자끼리 싸우던 표시를 "견적서 연결 전"
   상태 + "수금 N만원 기록됨" 안내로 교체.
4. **PC 사이드바 '전체 보기' 저대비 수정** — sticky 규칙의 배경이 active
   (파란 배경·흰 글자)를 덮어 흰 글자만 남던 CSS 우선순위 버그.
5. **중복 제목 제거** — 일정표·견적 작성 탭에서 화면 제목이 두 번 나오던
   것(상단 타이틀 + 패널 h2) 정리. 폰 세로 공간 절약.
6. **대시보드 퀵칩 정리** — 중복 "브리핑" 칩 제거(아침 자동 브리핑으로
   충분), 색상 위계 정리(주역 2개만 색, 나머지 ghost).
7. **모바일 헤더 축소** — 첫 화면의 ~22%를 차지하던 브랜딩 영역을 로고+
   역할 배지 한 줄로(슬로건은 폰에서 숨김).
8. **아이콘 버튼 접근명** — 가⁺/🌙/📱 버튼에 aria-label.
