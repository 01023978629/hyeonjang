# 변경사항 v108 - 방문 묶음 사진 복구

날짜: 2026-07-17

## 수정 내용

- 미리보기가 비어 있는 방문 묶음 헤더에 `사진 연결 N` 버튼을 추가했습니다.
- 버튼 한 번으로 해당 방문의 원본 사진을 여러 장 선택할 수 있습니다.
- 원본 파일명이 유지된 사진은 기존 기록과 자동 연결합니다.
- 파일명이 바뀌었더라도 선택 수와 빈 사진 수가 같으면 확인 후 선택 순서대로 연결합니다.
- 확인 전에는 어떤 사진도 변경하지 않으며, 사진 수가 다르면 다시 선택하도록 안내합니다.
- 연결할 때 기존 촬영일, 현장, 공정, 작업명은 그대로 보존합니다.
- 연결된 사진은 기존과 동일하게 썸네일 캐시 및 Apps Script Drive 업로드를 적용합니다.
- 서비스 워커 캐시 키와 화면 빌드 표기를 v108로 올렸습니다.

## 검증 항목

- 방문 묶음 복구 버튼 렌더
- 파일명이 다른 사진의 확인 전 무변경
- 선택 순서 연결 후 현장·공정 보존
- 로컬 썸네일 캐시와 relay 업로드 회귀
- 390px 및 430px 모바일 레이아웃
