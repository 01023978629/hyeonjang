현장 개인정보처리방침

시행일: 2026년 6월 25일

현장은 사용자가 선택한 Google Drive 파일과 앱이 생성한 현장 자료를 사용자의 업무 기기와 Google Drive에서 관리하는 도구입니다.

1. 처리하는 정보

  • Google 계정 OAuth 인증 정보 및 일시적인 액세스 토큰
  • 사용자가 앱에서 선택하거나 생성한 현장명, 일정, 견적, 연락처, 사진 및 문서
  • Google Drive에서 사용자가 이 앱으로 열거나 이 앱이 생성한 파일 정보

2. Google 사용자 데이터의 이용 목적

Google Drive 연결, 앱 데이터 백업·복원, 사용자가 선택한 파일 불러오기, 현장 사진·견적서·데이터 파일 저장에만 이용합니다. 앱은 필요한 범위를 줄이기 위해 기본적으로 drive.file 권한을 사용합니다. 사용자가 ‘내 드라이브 전체 통합 불러오기’를 직접 선택한 경우에 한해, 사용자의 만물인테리어 현장 사진·견적서를 화면에 표시하기 위해 읽기 전용 drive.readonly 권한을 사용하며, 읽은 파일은 사용자의 기기에서 보기 용도로만 쓰고 외부로 전송하지 않습니다. Google Sheets 기능을 이용할 때에는 사용자가 요청한 스프레드시트 생성·수정에만 Sheets 권한을 사용합니다.

3. 저장 및 공유

앱의 기본 데이터는 사용자의 브라우저 저장소와 사용자의 Google Drive에 저장됩니다. 운영자는 Google 사용자 데이터를 광고에 사용하거나 판매하지 않으며, 사용자의 명시적 요청 또는 법적 의무가 없는 한 제3자에게 제공하지 않습니다.

4. 보관 및 삭제

브라우저에 저장된 데이터는 사용자가 브라우저 데이터 삭제 또는 앱 초기화를 통해 삭제할 수 있습니다. Google Drive에 저장된 파일은 사용자가 Drive에서 직접 삭제할 수 있습니다. Google 계정의 연결 권한은 Google 계정 권한 관리에서 언제든 철회할 수 있습니다.

5. 보안

OAuth 액세스 토큰을 공개 저장소에 저장하지 않으며, HTTPS 통신을 사용합니다. 사용자는 공용 기기에서 사용 후 Google 연결을 해제하고 브라우저 데이터를 삭제해야 합니다.

6. Google API 정책 준수

현장의 Google API 정보 사용 및 다른 앱으로의 전송은 제한적 사용 요건을 포함한 Google API Services User Data Policy를 준수합니다.
