# 변경사항 v106 - 모바일 사진 미리보기 복구

날짜: 2026-07-17

## 수정 내용

- 모바일 중계 서버로 올린 사진을 Google 로그인 없이 미리볼 수 있습니다.
- Apps Script가 Drive 썸네일을 인증해서 전달하고, 앱은 받은 이미지를 IndexedDB에 캐시합니다.
- 앱을 닫았다 다시 열거나 다른 모바일 기기에서 서버 자료를 불러와도 사진 카드와 크게 보기가 동작합니다.
- 설정 화면의 사진 올리기도 업로드한 사진을 즉시 앱 목록에 연결합니다.
- 과거 모바일 중계 업로드분이 JSON에 연결되지 않았어도 서버 사진 목록에서 자동 복구합니다.
- 같은 파일명의 기존 빈 사진 기록에는 Drive ID를 직접 연결해 원래 위치에서 미리보기를 복구하고 중복 추가를 줄입니다.
- 썸네일 요청은 설정된 앱 루트 폴더 내부 파일만 허용하며, 응답 크기를 제한합니다.
- 서비스 워커 캐시 키와 화면 빌드 표기를 v106.2로 올렸습니다.

## 검증

- Apps Script, 모의 서버, 앱 인라인 스크립트 구문 검사 통과
- relay 저장, 충돌, 오프라인 큐, 사진/문서 업로드, 모바일 레이아웃 회귀 테스트 통과
- 390px 모바일 컨텍스트에서 Google 로그인 토큰 없이 사진 data URL 수신, 이미지 디코딩, 캐시 저장 확인

## 배포 시 주의

`index.html`만 배포하면 안 됩니다. `apps-script/Code.gs`를 Apps Script에 반영하고 기존 웹앱 배포를 새 버전으로 갱신해야 합니다.
