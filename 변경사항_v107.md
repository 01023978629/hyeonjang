# 변경사항 v107 - 빈 사진 미리보기 복구

날짜: 2026-07-17

## 수정 내용

- `IMG`로만 보이던 빈 사진 카드를 `사진 연결` 버튼으로 바꿨습니다.
- Apps Script 사진 목록 조회가 `현장사진` 폴더와 앱 루트의 하위 폴더를 재귀 탐색합니다.
- 파일명 뒤의 `의 사본`, `복사본`, `copy`, `(1)` 표기가 달라도 기존 사진 기록과 연결합니다.
- 루트 전체에서 찾은 명함·도면 이미지는 빈 기록 복구에만 사용하고 새 현장사진으로 잘못 추가하지 않습니다.
- 상단 `미리보기 복구`에서 모든 빈 사진을 한 번에 서버와 다시 연결합니다.
- 서버에 원본이 없으면 기기 사진을 여러 장 선택해 파일명으로 기존 기록과 연결할 수 있습니다.
- 다시 선택한 사진은 서버 중계가 설정돼 있으면 Drive에도 업로드하며, 같은 기기의 IndexedDB에도 썸네일을 보관합니다.
- 서비스 워커 캐시 키와 화면 빌드 표기를 v107로 올렸습니다.

## 검증 항목

- 파일명 정규화 자동 연결
- 빈 사진 카드 복구 버튼
- 기기 사진 재선택과 로컬 썸네일 캐시
- Google 로그인 없는 relay 썸네일 수신
- 390px 모바일 레이아웃과 기존 저장·복원 회귀
