# 📱 현장. 실기기 검증 체크리스트

배포 후 실제 폰에서 확인하는 순서. **안드로이드(크롬)**과 **아이폰(사파리)** 각각 한 번씩.
소요 약 15분. GitHub Pages 주소(`https://<사용자>.github.io/<저장소>/`)를 폰에서 연 상태로 시작.

---

## 1. 설치 (홈 화면 추가)

**안드로이드 (크롬)**
- [ ] 주소를 열면 하단/메뉴에 "앱 설치" 또는 "홈 화면에 추가" 배너가 뜬다
- [ ] 설치 후 홈 화면 아이콘으로 열면 **주소창 없이 전체화면**으로 뜬다
- [ ] 아이콘을 길게 누르면 바로가기(오늘 일정 / 미수금 / AI 비서 / 한눈 보기)가 나온다

**아이폰 (사파리)**
- [ ] 공유 버튼 → "홈 화면에 추가" 가 동작한다
- [ ] 홈 아이콘으로 열면 전체화면(상단 시간·배터리만)으로 뜬다
- [ ] 앱 이름이 "현장"으로 표시된다

> 문제 시: 매니페스트/서비스워커 문제. `sw.js`와 `manifest`가 같은 경로에 있는지, HTTPS인지 확인.

## 2. 첫 실행 / 온보딩

- [ ] 처음 열면 **3장 안내 슬라이드**가 뜬다(운영·AI·안전판)
- [ ] "다음"으로 넘기고 "시작하기" → **시작 준비 체크리스트**가 뜬다
- [ ] "예시로 둘러보기"를 누르면 데모 데이터가 들어간다
- [ ] 앱을 껐다 다시 열면 온보딩이 **다시 뜨지 않는다**

## 3. 핵심 동작 (오프라인 아님)

- [ ] 대시보드에 KPI(총 견적·수금·미수금)가 숫자로 보인다
- [ ] [📊 경영 현황] → 이번 달 요약 화면이 뜬다
- [ ] 사진 탭에서 사진이 묶음으로 보이고, 스크롤이 매끄럽다
- [ ] 현장 하나를 눌러 파일철(타임라인)이 열린다
- [ ] 하단 탭 전환이 즉각 반응한다(지연·깜빡임 없음)

## 4. AI 비서 (Gemini 키 입력 후)

- [ ] ☁️ 설정에서 무료 Gemini 키(AIza…)를 넣으면 저장된다
- [ ] [🤖 AI 비서]에 "오늘 뭐 챙겨야 해?" → 도구를 실행해 답한다
- [ ] "이번 주 리포트 만들어줘" → 주간 리포트 화면이 열린다
- [ ] 변경 지시("○○ 수금 100만원") → **승인 카드**가 뜨고, 승인하면 반영된다
- [ ] ⚡ 자동 실행을 켠 뒤 복합 지시("현장 만들고 실측 잡아줘") → 여러 단계가 자동 처리된다

## 5. 저장 / 데이터

**안드로이드(크롬, 폴더 지원)**
- [ ] [📁 폴더 연결] → 폴더 선택 후 자동 저장된다
- [ ] 앱을 껐다 켜도 자료가 남아 있다

**아이폰(사파리, 폴더 미지원)**
- [ ] [☁️ 드라이브] 또는 [📤 내보내기]로 백업이 된다
- [ ] 🛡 안전판(더보기)에 스냅샷이 쌓인다 → 복구가 동작한다
- [ ] [📥 데이터 이사]로 엑셀·백업 가져오기가 열린다

## 6. 카메라 / 사진

- [ ] 사진 추가에서 **카메라 촬영**이 바로 열린다(권한 허용)
- [ ] 여러 장 선택 업로드가 된다
- [ ] HEIC(아이폰 기본) 사진도 미리보기가 뜬다
- [ ] 사진 GPS가 있으면 자동 정리 제안에 반영된다

## 7. 공유 / 문자

- [ ] 진행 보고 [✉️ 고객에게 문자] → **문자 앱**이 수신자·내용 채워져 열린다
- [ ] 주간 리포트 [이미지] → 카톡 등으로 **공유 시트**가 뜬다
- [ ] 고객 진행 페이지 [만들기] → HTML 파일 공유가 된다
- [ ] 견적서 PDF가 생성·공유된다

## 7.5 알림·장부·주간 브리핑 (v20 신기능)

**스마트 알림**
- [ ] 더보기 [🔔 스마트 알림] → [알림 켜기] 누르면 권한 요청이 뜬다
- [ ] 허용 후 [지금 테스트] → 오늘 일정·미수·A/S 만료가 **알림으로 뜬다**
- [ ] 앱을 껐다 다음 날 열면 하루 한 번 알림이 온다(중복 안 뜸)
- [ ] 알림을 탭하면 앱이 열리고 경영 대시보드가 뜬다

**간편 지출 장부**
- [ ] 더보기 [💳 간편 장부] → [➕ 지출 추가]로 금액·분류·현금/카드 입력이 된다
- [ ] 저장하면 목록에 뜨고 월 총액·현금/카드 집계가 갱신된다
- [ ] 비서에게 "오늘 자재비 5만원 카드로 썼어" → 지출이 기록된다
- [ ] 월 이동(‹ ›)과 [📊 이 달 엑셀]이 동작한다

**AI 주간 브리핑**
- [ ] 월요일에 앱을 열면 🤖 주간 브리핑이 자동으로 뜬다
- [ ] 경영 대시보드 [🤖 주간]으로 언제든 열린다
- [ ] Gemini 키가 있으면 [🤖 AI 코멘트]가 조언을 붙여준다

## 7.6 AI 자율 운영 (v21~v28 신기능 — 실기기 필수 확인)

**AI 비서 능동 운영**
- [ ] 비서를 열면 상단에 "지금 하면 좋은 일" 제안이 뜬다(완료 현장 청구서·미배정 사진 등)
- [ ] 제안/빠른 명령 버튼을 누르면 바로 실행된다
- [ ] "이번달 얼마 남았어" → 손익, "못 받은 돈 점검" → 미수금이 열린다
- [ ] 상단 통합 검색으로 현장·사진·메모·거래처가 한 번에 찾아진다

**돈·목표·예산 (수금 기준)**
- [ ] "오늘 자재비 5만원 카드로 썼어" → 지출이 기록되고 월 손익에 반영된다
- [ ] "이번달 목표 1000만원으로 잡아줘" → 목표가 설정되고 달성률이 표시된다
- [ ] "둔산동 예산 800만원으로" → 예산 대비 지출·초과 경고가 뜬다
- [ ] 경영 대시보드가 💰돈/📊리포트/🗂현장 카테고리로 정리돼 있다

**🎙 음성 (실기기에서만 정확히 확인 가능)**
- [ ] 더보기 [📝 음성 현장일지] → [녹음 시작] 후 말하면 글로 적힌다
- [ ] 저장하면 해당 현장 작업일지로 기록된다
- [ ] (음성 인식은 크롬/사파리 등 최신 브라우저에서 동작)

**📷 카메라·사진 AI (실기기 필수)**
- [ ] 사진을 찍거나 올려 현장에 배정된다
- [ ] 더보기 [📷 사진 AI 분석] → 사진 선택 → 공정·하자 판단이 나온다(Gemini 키 필요)
- [ ] "미배정 사진 정리해줘" → 날짜·현장별로 자동 정리된다

**💸 AI 견적·감리 (Gemini 키 필요)**
- [ ] 더보기 [💸 AI 견적작성] → 현장 설명 입력 → 견적 초안이 항목별로 생성된다
- [ ] "32평 전체 도배하고 바닥 장판 견적 만들어줘" → 견적 탭에 초안이 만들어진다
- [ ] [📈 경영 분석]에서 이익률·미수금 인사이트가 나온다

**📅 일정 브리핑**
- [ ] 더보기 [📅 일정 브리핑] → 오늘 일정이 시간순+주소+준비물과 함께 뜬다
- [ ] 주소를 누르면 지도 앱으로, 전화번호를 누르면 통화로 연결된다

**📚 사용 가이드**
- [ ] 더보기 [📚 사용 가이드] → 7개 분야별로 기능과 "말하는 예시"가 보인다
- [ ] 예시의 [🤖 비서에게 시키기]를 누르면 비서에 자동 입력된다

## 7.7 최신 AI 기능 (v29~v34 — 실기기 최종 확인)

**🔑 AI 제공자 (Gemini 무료 / OpenAI 선택)**
- [ ] ☁️ 설정에 Gemini 키(무료) 또는 OpenAI 키(sk-)를 넣을 수 있다
- [ ] 두 키가 다 있으면 무료 Gemini를 우선 사용한다(비용 0)

**🤖 AI 능동 운영**
- [ ] 앱을 켜면 하루 한 번 "오늘의 브리핑"이 뜬다(오늘 일정·미수·챙길 일)
- [ ] 브리핑의 바로 처리 버튼을 누르면 해당 작업이 실행된다
- [ ] 더보기 [⚙️ AI 자동 모드]에서 안전 작업 자동 실행을 항목별로 켜고 끌 수 있다
- [ ] 사진 정리 같은 안전한 일은 승인 없이, 삭제·발송은 확인을 거친다

**📊 운영 리포트·예측**
- [ ] 경영 대시보드 [📊 운영]에서 주간/월간 운영이 정리된다
- [ ] 일요일·월말에 앱을 켜면 운영 리포트가 자동으로 뜬다
- [ ] 경영 대시보드 [📈 예측]에서 다음달 매출 예측이 나온다

**💬 고객 응대·문서 AI**
- [ ] 더보기 [💬 고객 문자] → 상황 선택 → 문자 초안이 만들어지고 문자앱으로 전송된다
- [ ] 견적 [📜 계약] → [🤖 AI로 공사범위·특약 작성]이 계약서 특약을 채운다
- [ ] 계약서에 손가락으로 서명이 그려진다 (실기기 필수)
- [ ] 더보기 [📇 세금계산서] → 품목 [🤖 AI]가 품목을 정리한다

**🔄 PC ↔ 폰 동기화 (실기기 필수)**
- [ ] 더보기 [🔄 PC↔폰 동기화]에서 사용법을 볼 수 있다
- [ ] PC에서 [☁️ 올리기] → 폰에서 [☁️ 전체 불러오기]로 PC 작업이 폰에 나타난다
- [ ] 같은 구글 계정으로 양쪽에서 로그인된다

## 8. 오프라인 강건성 (비행기 모드 켜고)

- [ ] 앱이 **열린다**(캐시된 셸)
- [ ] 하단에 **오프라인 배너**가 뜬다("기록·조회는 그대로")
- [ ] 현장·수금 등 **기록·조회는 정상 동작**한다
- [ ] 지도/엑셀/PDF 버튼을 누르면 **"연결 필요" 안내**가 뜬다(조용히 실패 안 함)
- [ ] 비행기 모드 해제 → "다시 연결됐어요" 토스트가 뜬다

## 9. 브랜드 / 마무리

- [ ] 더보기 [🎨 브랜드 테마]에서 색·로고를 바꾸면 **앱 상단·리포트**에 반영된다
- [ ] 화면 회전(가로/세로)해도 레이아웃이 깨지지 않는다
- [ ] 다크 모드(기기 설정)에서 글자가 읽힌다

## 흔한 문제 → 원인

| 증상 | 확인할 것 |
|---|---|
| 홈 추가 배너 안 뜸 | HTTPS인지, 매니페스트 로드되는지(개발자도구 Application) |
| 껐다 켜면 자료 사라짐 | 폴더/드라이브 연결 여부 — 미연결 시 이 기기 스냅샷만 보관 |
| AI가 답 안 함 | Gemini 키 유효한지, 일일 무료 한도 초과 아닌지 |
| 사진 안 열림 | HEIC 변환 실패 가능 — 다른 사진으로 재시도, 네트워크 확인 |
| 문자/공유 안 됨 | 기기 기본 앱 설정, 사파리는 사용자 제스처 필요 |
| 지도 안 뜸 | 오프라인이거나 CDN 차단 — 온라인에서 재시도 |

*체크 항목이 모두 통과하면 실사용 배포 준비 완료.*
